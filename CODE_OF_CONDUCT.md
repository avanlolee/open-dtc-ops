# Code of Conduct

We want `open-dtc-ops` to be useful and welcoming for small ecommerce operators and contributors.

## Expected Behavior

- Be respectful and constructive.
- Explain tradeoffs clearly.
- Help beginners understand practical ecommerce operations.
- Keep discussions focused on the project.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private business data.
- Pushing features that violate platform terms or automate unauthorized access.

## Reporting

Please report concerns through a GitHub issue or by contacting the maintainers.
