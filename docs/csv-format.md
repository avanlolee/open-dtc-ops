# CSV Format

## Sales CSV

Required columns:

```text
date,product,revenue,ad_spend,cogs,shipping,platform_fee,units_sold
```

## Inventory CSV

Required columns:

```text
product,current_inventory,avg_daily_sales,lead_time_days,safety_stock
```

## Product Research CSV

Required columns:

```text
product,demand_score,margin_score,competition_score,pain_point_score,differentiation_score
```

Scores should use a 1 to 10 scale.
